R syntax

General syntax info:

  • R is case sensitive
  • spaces are ignored by R
  • there is no difference between single ' and double " quotes but you have to use the same type to close as you open with

Mathematical operators

addition: +

subtraction: -

multiplication: *

division: /

exponent: **

modulo/remainder: %%

# addition
1 + 2
#> [1] 3

# subtraction
3 - 6
#> [1] -3

# multiplication
4 * 2
#> [1] 8

# division
12 / 3
#> [1] 4

# exponent
2 ** 5
#> [1] 32

# modulo
5 %% 2
#> [1] 1

Expanding R’s functionality

You can add packages to R using install.packages("<package name>") - you only need to install a package once1. Once a package is installed, each time R is opened you will need to use library(<package name>) to make the package available for the session.

Base R and Tidyverse

install.packages("tidyverse")
library(tidyverse)

  1. packages need to be re-installed for major R version upgrades↩︎
